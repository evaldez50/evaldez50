## 🚀 Proyectos

### [Le Pupu Le Guagua](https://github.com/evaldez50/le-pupu-le-guagua)
App web para aprender francés desde español. Ejercicios interactivos por niveles (A1–C2), conjugador de verbos, audio con TTS, modelo freemium con Stripe y auth con Google OAuth via Supabase.

### [Voice Notes AI](https://github.com/evaldez50/voice-notes-ai)
App full-stack para grabar notas de voz, transcribirlas con Whisper y analizarlas con Claude. Chat con contexto de grabaciones, resúmenes ejecutivos y mapas mentales generados con AI.

### [BrainNotes](https://github.com/evaldez50/brainnotes-cloud)
API y UI para capturar notas de voz y hacer consultas semánticas usando OpenAI vector stores y GPT-4.1-mini con file search. Incluye [agente de ingesta automática](https://github.com/evaldez50/brainnotes-agent).

### [Claude Learning Tracker](https://github.com/evaldez50/claude-tracker)
PWA para rastrear progreso en el plan de aprendizaje de Claude/Anthropic. Roadmap de 14 semanas, 13 módulos, 7 certificados oficiales. React + Vite + Tailwind, offline-first con PWA.
